?? FLUXO DE DADOS

Frontend (index.html)
        ?
POST /api/auth/login { email, password }
        ?
AuthController.Login()
        ?
AuthService.HashPassword() / VerifyPassword()
        ?
AppDbContext.Users.FirstOrDefault()
        ?
AuthService.GenerateJwtToken()
        ?
Return { token, user }
        ?
Frontend armazena em localStorage
        ?
GET /api/auth/profile
    + Authorization: Bearer <token>
        ?
Middleware valida JWT
        ?
[Authorize] permite acesso
        ?
Return Perfil do usuário

?? FLUXO GOOGLE OAUTH

Frontend (Google Sign-In)
        ?
GoogleClientLibrary abre dialog
        ?
Usuário autoriza
        ?
Google retorna idToken
        ?
Frontend envia:
POST /api/auth/google-login { idToken }
        ?
AuthController.GoogleLogin()
        ?
AuthService.VerifyGoogleTokenAsync()
        ?
GoogleJsonWebSignature.ValidateAsync()
        ?
Valida assinatura e expiração
        ?
Retorna GoogleJsonWebSignature.Payload
        ?
Procura usuário por GoogleId
        ?
Se não existe: cria novo
        ?
AuthService.GenerateJwtToken()
        ?
Return { token, user }
        ?
Frontend armazena e redireciona
