?? PÚBLICOS (sem autenticação):

  POST /api/auth/register
    Body: { email, password, name }
    Response: { token, user }

  POST /api/auth/login

    Body: { idToken }
    Response: { token, user }

?? PROTEGIDOS (requer JWT):

  GET /api/auth/profile [Authorize]
    Headers: Authorization: Bearer <token>
