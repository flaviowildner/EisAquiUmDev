? FLUXO COMPLETO IMPLEMENTADO

1?? USUÁRIO CLICA GOOGLE
   ??> Google Sign-In JavaScript abre dialog

2?? GOOGLE RETORNA ID TOKEN
   ??> Token JWT assinado pelo Google

3?? FRONTEND ENVIA PARA BACKEND
   ??> POST /api/auth/google-login { idToken }

4?? BACKEND VALIDA TOKEN
   ??> GoogleJsonWebSignature.ValidateAsync()

5?? BACKEND PROCURA/CRIA USUÁRIO
   ??> Verifica GoogleId ou cria novo

6?? BACKEND GERA JWT DA API
   ??> Token assinado com sua SecretKey

7?? RETORNA JWT PARA FRONTEND
   ??> Response: { token, user }

8?? FRONTEND ARMAZENA EM localStorage
   ??> Guarda token para requisiçőes futuras

9?? FRONTEND REDIRECIONA
   ??> Para dashboard/perfil

?? TODAS AS REQUISIÇŐES USAM JWT
